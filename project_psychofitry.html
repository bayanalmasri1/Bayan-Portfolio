Project Overview

Psychofitry is a mental health mobile application concept built in Flutter to make psychological support easier to access through a calm interface and structured user journey. The app combines onboarding, account creation, login, verification, educational content, doctor information, media content, profile management, and support-oriented settings.

The project demonstrates thoughtful Arabic RTL interface design, healthcare-oriented navigation, feature grouping, and a multi-screen product flow shaped for trust, clarity, and ongoing engagement.

Key Features

  • Arabic-first authentication flow with signup, login, and OTP verification.
  • Content-driven home experience featuring books, videos, and educational topics.
  • Doctor detail screens with specialization, experience, and scheduling context.
  • Profile hub covering personal data, medical file, favorites, payment, and help center access.
  • Dedicated settings screens for notifications, language, theme, and account actions.
