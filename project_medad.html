Project Overview

Medad is a complete school management system developed with Flutter. I worked on three related mobile applications within the system, covering administration workflows, teacher app features, account management, delete account requests, and school-related processes.

The project demonstrates experience in multi-app architecture, API integration, role-based flows, reusable UI components, and production-ready Flutter development for structured operational use cases.

Key Features

  • Three Flutter mobile applications within one school system.
  • Admin app features for account management and deletion requests.
  • Teacher app workflows and school-related functionality.
  • Role-based user experience across multiple user types.
  • REST API integration with maintainable mobile architecture.
  • Arabic and English support with RTL-ready UI where needed.
