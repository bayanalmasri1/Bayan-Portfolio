Project Overview

Midadd Teachers App is a Flutter mobile application developed as part of the Midadd school management platform. The app provides teachers with the tools they need to manage their daily school workflows efficiently, including classroom management, student attendance, grades, assignments, assessments, schedules, notifications, and access to students’ academic and behavioral records.

I worked on teacher-focused features and school workflows within the Midadd ecosystem, with attention to API integration, reusable Flutter UI components, role-based flows, Arabic support, RTL-ready layouts, and production-ready mobile development.

Key Features

  • Teacher login and secure access.
  • Daily classes and schedule overview.
  • Student attendance and absence reporting.
  • Student list and student profile details.
  • Homework, assignments, worksheets, and evaluations.
  • Grade entry and result upload workflows.
  • Notifications and school updates.
  • Teacher requests and administrative notes.
  • Arabic language support and RTL-ready UI.
  • REST API integration with maintainable Flutter structure.
