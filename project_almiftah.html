Project Overview

Al Miftah is a real-estate mobile application developed with Flutter as freelance work for Rodex/Codex. The app focuses on helping users browse properties, view listing details, search and filter available options, and contact the relevant party easily.

The project demonstrates experience in real-estate app flows, REST API integration, responsive Flutter UI, reusable widgets, and clean mobile architecture.

Key Features

  • Property listing browsing.
  • Property details screen.
  • Search and filtering functionality.
  • Clean and responsive Flutter UI.
  • REST API integration.
  • Reusable components.
  • Real-estate customer experience focused design.
