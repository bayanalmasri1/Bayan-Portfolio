Project Overview

This project was built as a mobile flower shop application where users can discover products, explore pricing and variations, and complete purchases through a smooth shopping flow.

It demonstrates experience in product presentation, Firebase-backed authentication, interactive UI design, and commerce-oriented user journeys built with Flutter.

Key Features

  • Product catalog displaying flower details such as type, color, and price.
  • User authentication flow for personalized shopping experiences.
  • Shopping cart and checkout journey for streamlined ordering.
  • Responsive layouts tailored to Android and iOS devices.
  • Firebase integration for data and account handling.
