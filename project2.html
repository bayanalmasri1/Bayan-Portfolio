Flora Haven

Built a Flutter flower shop app with Firebase backend, payment integration, and admin controls, improving sales efficiency by 30%.

June 2025 Flutter , Firbase

Project Overview

Independently developed a Flutter application displaying flowers with their prices, colors, and types, including a payment and purchase system connected to a database.

Key Features

  • Product Catalog: Displays flowers with details (price, color, type, and availability) in an interactive UI
  • Shopping Cart & Checkout: Integrated a seamless cart system with secure payment processing (Stripe/PayPal).
  • User Authentication: Email/Google login with Firebase Auth for personalized experiences.
  • Responsive Design: Optimized for both iOS and Android with smooth animations.
  • Technologies: Flutter, Dart, Firebase (Auth, Firestore).
