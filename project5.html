Project Overview

Task Manager is a lightweight mobile app built with Flutter to support everyday organization through a clear workflow for adding, reviewing, completing, and deleting tasks.

The project demonstrates attention to usability, simple interaction patterns, and a clean UI approach for productivity-focused apps.

Key Features

  • Welcome screen introducing the app with a clear visual direction.
  • Task list interface for viewing and managing daily items.
  • Mark tasks as complete and remove tasks that are no longer needed.
  • Add new tasks through a popup dialog with title and description fields.
  • Clean interface with responsive behavior and subtle interactions.
