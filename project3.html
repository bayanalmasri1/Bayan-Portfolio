Project Overview

This mobile marketplace was designed to showcase Apple products such as iPhones and MacBooks in a clean, modern interface with product details, filters, and a simplified user path from browsing to purchase.

The project reflects strong attention to UI quality, shopping flow design, and crafting a polished mobile e-commerce experience with Flutter.

Key Features

  • Product catalog with detailed item displays and visual hierarchy.
  • Category and feature-based filtering for easier product discovery.
  • Checkout and order completion screens for a clear shopping funnel.
  • Onboarding, login, OTP, and account-related user journey states.
  • Smooth UI transitions and refined presentation of Apple products.
