Wow Syria App

Flutter app for property/vehicle listings with WhatsApp contact, built with GetX & HTTP (published on Google Play).

June 2023 Flutter

Project Overview

Developed a mobile application for “Sawa Company” published on google play. This app allows users to post their own ads for properties(apartments with details like area, location, bedrooms, price) and vehicles (cars / trucks with color, model, year, price, location) for sale or rent across Syrian cities. Users can contact advertiser via WhatsApp directly from the app. Implemented Getx for state management and used the HTTP library for API integration

Key Features

  • Platform (Google Play)
  • Core functionality (property/vehicle listings)
  • User reviews and ratings system
  • Key features (detailed ads, WhatsApp contact)
  • Technical stack (Flutter, GetX, HTTP)
