Published marketplace experience covering residential and vehicle ads across Syrian cities.

Project Overview

Wow Syria was developed for a local company as a mobile platform where users can publish and explore real estate and vehicle listings for sale or rent. The app handles detailed item data, browsing by location, and quick communication with advertisers.

The project highlights experience in mobile publishing flows, API integration, scalable state handling with GetX, and user-friendly listing experiences tailored to regional users.

Key Features

  • Property and vehicle listing flows with structured detail fields.
  • Location-aware browsing across multiple Syrian cities.
  • Direct WhatsApp communication with advertisers.
  • Dynamic data handling powered by HTTP API integration.
  • Responsive mobile UI with practical publishing and discovery experience.
